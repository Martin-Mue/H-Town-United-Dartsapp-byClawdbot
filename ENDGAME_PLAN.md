# Endgame-Plan: Beste Darts App

Stand: 2026-03-01

## Phase A — Kamera (Dartsmind-Level)
1. Auto-Loop produktiv (Detect -> Remove -> Commit)
2. Board-Kalibrierung robust + Qualitätsstufen + Recalibrate
3. Visit-Erkennung (1–3 Darts) stabil unter Realbedingungen
4. Remote-Livefeed (WebRTC) vollständig inkl. Reconnect/Latenzstatus
5. Confidence-/Fallback-Regeln + Manual Review ohne Datenverlust

## Phase B — Spielsession & UX
6. Harte Active-Match-Session-Logik (bereits gestartet)
7. Setup-Wizard (3 Schritte, klare Validierung)
8. Intuitive Match-Steuerung (Pause/Resume/Ende/Abbruch)

## Phase C — Statistik (DartCounter-Level+)
9. KPI-Harmonisierung überall (Match/Profil/Statistik/Training)
10. Tiefere Reports + Exporte + Vergleich/Heatmaps final polieren

## Abnahmekriterien
- Komplettes Match ohne manuelle Punkteingabe via Kamera möglich
- Keine konkurrierenden Match-Sessions
- Mobile-First Flow ohne Sackgassen
- KPI-Werte auf allen Screens konsistent
